## 🚀 使用方法

### 基本使用
1. 运行 `python main.py` 启动程序
2. 输入YouTube频道URL
3. 点击"解析频道"获取视频列表
4. 选择要下载的视频（使用复选框）
5. 选择视频质量
6. 点击"开始下载"

### 选择功能
- **全选**: 选中所有视频
- **取消全选**: 取消所有选择
- **反选**: 反向选择当前状态
